Guide d'Utilisation de NATYABIP

Rédaction : NATYABIP

Version : 1 - 07/01/2026

Site/Portail : portail.natyabip.com

Ce guide détaille l'utilisation de la plateforme NATYABIP SMS, couvrant la gestion du tableau de bord, des profils, des comptes, l'envoi de SMS, les rapports et les campagnes, pour une prise en main complète et efficace.

I. Tableau de Bord

Le tableau de bord présente un résumé de la situation du jour et du mois en fonction du compte sélectionné.

Fonctionnalités Clés

  • Solde Total : Affiche le solde global associé au compte connecté.
  • Espace Profil : Accès à la section Profil, déconnexion, gestion du mot de passe et tarification.
  • Situation du Jour : Informations basées sur la date actuelle.
  • Situation du Mois : Intervalle complet du mois en cours ou période sélectionnée.
  • Statistiques : Statistiques des messages, taux et indicateurs clés.

II. Profil

Consultation du Profil

Vous pouvez visualiser votre profil contenant vos informations de base (Initiales, Nom, Email, Seuil d'alerte, etc...).

Modification des Informations

Dans l'espace de modification de profil, vous pouvez gérer les paramètres suivants :

  • Société : Nom de l'entreprise associée au compte.
  • Nom : Nom de famille de l'utilisateur.
  • Prénom : Prénom de l'utilisateur.
  • Email : Votre adresse électronique de contact.
  • Numéro Cellulaire : Modifier votre numéro de téléphone.
  • Expéditeur : Le nom d'expéditeur par défaut.
  • Seuil d'alerte : Niveau de crédit pour déclencher une alerte.
  • Long SMS : Activation de l'envoi de SMS longs (concaténés).
  • Type d'Encodage : Encodage des caractères (ex: GSM7).
  • Mode de facturation : Type de paiement (Prépayé ou Postpayé).
  • Etat du compte : Situation actuelle du compte (Actif/Suspendu).
  • Montant découvert : Limite de crédit autorisée en cas de découvert.

Cliquez sur Enregistrer les modifications pour valider vos changements.

III. Comptes

Historique Balance

Sélectionnez le compte concerné, choisissez la plage de dates (début/fin), puis cliquez sur Rechercher pour afficher l’historique de balance.

Solde

Sélectionnez le compte concerné, puis cliquez sur Rechercher pour afficher le solde.

Relevé

Sélectionnez le compte concerné, puis cliquez sur Rechercher pour afficher le relevé de compte.

Expéditeur (Sender ID)

Cette zone permet de gérer les expéditeurs :

  • Consulter : Filtrer par compte ou pays.
  • Demande : Saisir le Sender ID souhaité puis cliquer sur VALIDER.

NB : Le traitement prend environ trois jours.

IV. Traitement SMS

Envoi Simple

Pour envoyer un SMS :

  1. Sélectionnez l’expéditeur.
  2. Saisir le(s) destinataire(s) (numéro avec indicatif, ex: 228XXXXXXXX). Séparez les numéros multiples par des points-virgules (;).
  3. Saisir le message (comportant un nombre de caractère en fonction du type d’encodage)
  4. Cliquez sur VALIDER.

V. Rapports

Consultez les rapports détaillés sur vos envois.

Filtres de Recherche

  • Compte / Pays / Expéditeur / Opérateur : Filtrez selon vos besoins.
  • Plage de Date : Définissez la période.
  • État : Filtrer par (Accepté, En cours, Envoyé, Délivré, Confirmé, Rejeté).

Utilisez le bouton Rechercher pour afficher les résultats ou Annuler pour réinitialiser.

Situation Journalière

Affiche la situation des SMS pour une journée donnée après sélection du compte et de la date.

VI. Carnet d'Adresse

Ajouter un Contact

Remplissez les champs Nom, Prénom, Indicatif (ex. 228) et Numéro. Vous pouvez assigner une catégorie (ex. Client fidèle) ou en créer une nouvelle. Cliquez sur Enregistrer.

Importer & Rechercher

Utilisez le bouton Importer pour charger des contacts depuis un fichier externe (Excel). Recherchez par nom ou numéro et filtrez par période.

VII. Campagne

Créer une Campagne

  1. Donnez un nom à votre campagne.
  2. Choisissez la destination et l'expéditeur.
  3. Planifiez la date et l'heure.
  4. Rédigez le message (comportant un nombre de caractère en fonction du type d'encodage) et importez les contacts (Excel).
  5. Simulez pour vérifier ou validez.

Détail

Consultez les détails de la campagne (date, contenu, état).

Liste

Filtrez et visualisez la liste de vos campagnes.

Modifier / Activer une Campagne

Pour modifier ou activer une campagne :

  1. Sélectionnez la campagne à modifier.
  2. Modifiez la date/heure ou le message si nécessaire.
  3. Changez l'état de la campagne :
    • Brouillon : État par défaut à la création.
    • Active : Nécessaire pour envoyer/transférer les messages.
    • Terminée / Annulée : Pour clore ou annuler la campagne.
  4. Enregistrez les modifications.

NB : L'état doit être sur Active pour que le message soit envoyé.
